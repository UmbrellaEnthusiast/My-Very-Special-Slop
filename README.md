My very special slop is a mod with vanilla balancing but non vanilla features

the mod is meant to feel like a dlc for a game that comes out years after the initial release

the mod introduces 2 new raritys "shinies" and "splendids"

shinies are rare but improved versions of jokers from the mod and splendids are like legendaries but are obtained like any other joker

the mod doubles the current amount of decks
